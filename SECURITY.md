# Security and Responsible Use

AZAI is intended for analytical chemistry, forensic/public-health research, fluorescent probe discovery, molecular characterization, and medicinal chemistry education.

Do not use AZAI to optimize xylazine potency, abuse potential, harmful delivery, clandestine synthesis, or illicit drug production. Issues or pull requests that attempt to redirect the project toward harmful use will be closed.

To report a security or misuse concern, open a private security advisory on GitHub when available.
