# Reporting and export bundles

AZAI v0.5.0 adds dependency-light scientific reporting utilities for sharing transparent cheminformatics outputs.

## Outputs

The export bundle contains:

- `AZAI_report.md` — human-readable Markdown report
- `AZAI_report.html` — standalone HTML report
- `descriptors.json` — machine-readable descriptor output
- `descriptors.csv` — spreadsheet-friendly descriptor output
- `interferent_risk.csv` — heuristic selectivity-risk comparison
- `SAFETY.txt` — allowed-use reminder

## Important limitation

Reports are computational summaries and prioritization aids. They are not experimental validation, forensic confirmation, clinical advice, or synthetic guidance.
